# Projet ECF entreprise "Vite & Gourmand"
Application web développée dans le cadre de la formation Développeur Web et Web Mobile STUDI.
Elle permet à l’entreprise "Vite & Gourmand" de présenter ses menus et de faciliter la prise de commande en ligne.

---

## Prérequis
- macOS / Windows
- XAMPP / MAMP installé
- PHP
- MySQL
- Navigateur web (Google Chrome recommandé)
- Git (pour cloner le projet)

---

## Récupération du projet
Cloner le dépôt GitHub :
git clone https://github.com/Mathieupalle/vite-gourmand.git

Ou télécharger le dossier en format ZIP depuis GitHub.

---

## Installation et déploiement en local
1. Placer le dossier du projet dans le dossier htdocs de XAMPP :
/Applications/XAMPP/htdocs/

2. Lancer XAMPP

3. Démarrer les modules :
- Apache
- MySQL

4. Créer la base de données via phpMyAdmin (à venir)
   
5. Accéder au projet via le navigateur :
http://localhost/vite-gourmand

---

## Gestion de version
Le projet est versionné avec Git et hébergé sur GitHub.
La gestion des versions est effectuée via le terminal (Bash sous macOS).
